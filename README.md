[# 🍔 Food Delivery Website

A full-stack **Food Delivery Web Application** built with **React.js, Node.js, Express.js, and MongoDB**. The application provides a seamless online food ordering experience with secure authentication, responsive design, real-time search, and shopping cart functionality.

## 🚀 Features

- 🔐 Secure user registration and login using JWT Authentication
- 🍽️ Browse and order from 100+ food items
- 🔍 Real-time food search with instant filtering
- 🛒 Shopping cart with:
  - Add/Remove items
  - Update item quantity
  - Order management
- 🛡️ Protected routes for authenticated users
- 📱 Fully responsive design for desktop, tablet, and mobile
- ⚡ Fast and interactive UI built with React Hooks

## 🛠️ Tech Stack

## ⚙️ Installation

### Clone the repository

```bash
git clone:https://github.com/somya-21/Deshi-Dash
cd food-delivery
```

### Install dependencies

#### Frontend

```bash
cd client
npm install
npm start
```

#### Backend

```bash
cd server
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the server directory and add:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```
